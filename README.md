## 特点

- 极简界面
- 纯文本界面，方便高级编辑
- 支持几乎全部数学函数
- 支持角度-弧度转换
- 支持引用数学常数：`π`（`pi`）、`e`、`tau`
- 单文件便携版

## 开发环境

1. Visual Studio 2022 (Community 即可, 64-bit)
    - .NET 桌面开发
        - NuGet 目标和生成任务
        - MSBuild

使用 `dotnet build` 以自主构建。
